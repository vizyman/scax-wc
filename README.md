# scax-wc

Lit + Three.js based web component library.

## Install

```bash
npm install scax-wc lit three
```

## Usage

```ts
import 'scax-wc';
```

```html
<scax-three-viewer></scax-three-viewer>
```

## Development

```bash
npm install
npm run check
npm run build
```

## Publish to npm

```bash
npm login
npm publish --access public
```
